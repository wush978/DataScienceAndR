# 說明

## 四小龍趨勢
農業, 輕工業 → 吸引資金與技術 + 本地勞動力(出口導向) → 東亞與東南亞主要經濟體( 1970- 1990 )

## 各自發展特色
- 香港 - 自由經濟 - 金融發展 + 自由貿易 - 無主導產業，也不設關稅
- 新加坡 - 政府對社經發展干預 → 外國投資者技術創新(42%外資投資)
- 台灣 - 外國投資者 + 在地企業合作 (加工出口區) 獲得技術
- 南韓 - 購買技術設備 → 模仿 → 改良 → 創新
